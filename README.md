# Gemmra Website

Landing page for **Gemmra — Nidāna for Pharmacovigilance**.

Built with [Astro](https://astro.build/).

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

Deploy `dist/` to Cloudflare Pages or any static host.
Site URL: https://gemmra.bhaskarjha.dev
